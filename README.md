## Currently Features

- [x] Show Local date and time
- [x] Show Crypto market pice from Coingecko API
- [ ] Task to Do list
- [ ] Expense Tracker

## Starting Development

Start the app in the `dev` environment:

```bash
yarn start
```

## Packaging for Production

To package apps for the local platform:

```bash
yarn package
```
